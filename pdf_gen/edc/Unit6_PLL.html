Block Diagram
Block Diagram of PLL
Introduction (1 Mark)

A Phase Locked Loop (PLL) mainly consists of the following three blocks –

  • Phase Detector
  • Active Low Pass Filter
  • Voltage Controlled Oscillator (VCO)

The output of a phase detector is applied as an input of active low pass filter. Similarly, the output of active low pass filter is applied as an input of VCO. The output of VCO is fed back to the second input of the phase detector, forming a closed loop.

Working of Each Block (3 Marks)
  • Phase Detector: It is a multiplier circuit. It compares the phase and frequency of the input signal (\(f_{in}\)) with the feedback signal from VCO (\(f_{out}\)). It produces a DC voltage that is directly proportional to the phase difference between these two signals. Its output contains two frequency components – the sum (\(f_{in} + f_{out}\)) and the difference (\(f_{in} - f_{out}\)) of the two input frequencies.
  • Active Low Pass Filter (LPF): It receives the output of the phase detector and removes the high-frequency sum component (\(f_{in} + f_{out}\)). Only the low-frequency difference component (a DC voltage) is allowed to pass through. This filter also amplifies the DC control voltage before feeding it to the VCO. The bandwidth of the LPF determines the capture range of the PLL.
  • Voltage Controlled Oscillator (VCO): It is a free-running oscillator that generates an output signal whose frequency is directly proportional to the input DC control voltage. When no input is applied, it oscillates at its natural free-running frequency (\(f_0\)). When a DC voltage from the LPF is applied, the VCO shifts its frequency to match the input signal. The frequency deviation is directly proportional to the DC voltage present at the output of the low pass filter.
Overall Working of PLL (2 Marks)

When an input signal \(f_{in}\) is applied, the phase detector compares it with VCO output \(f_{out}\). If there is any difference in phase or frequency, the phase detector generates an error voltage. This error voltage, after being filtered and amplified by the LPF, drives the VCO. The VCO adjusts its frequency in the direction that reduces the phase error.

This process continues until the VCO frequency becomes exactly equal to the input signal frequency (\(f_{out} = f_{in}\)). At this point, the PLL is said to be in a "locked" state. Once locked, the PLL tracks any small variations in the input signal frequency.

Q.2 Explain the characteristics of PLL 6 Marks
Introduction (1 Mark)

PLL operates in one of the following three modes depending on whether an input signal is applied and whether the loop has acquired lock:

  • 1. Free running mode
  • 2. Capture mode
  • 3. Lock mode
1. Free Running Mode (1.5 Marks)
  • When no input signal is applied to the PLL, the VCO operates at its own natural frequency called the free running frequency (\(f_0\)).
  • The output of the phase detector is zero because there is no input to compare with.
  • The DC control voltage to the VCO is zero, so the VCO runs independently.
  • The free running frequency is determined by the external timing components R1 and C1 connected to the VCO. The formula is: \( f_0 = \dfrac{0.3}{R_1 C_1} \)
2. Capture Mode (1.5 Marks)
  • When an input signal having some frequency is applied to PLL, the phase detector begins to compare the input frequency with VCO output.
  • The output signal frequency of VCO will start to change and move towards the input frequency.
  • At this stage, the PLL is said to be operating in the capture mode. The loop is trying to "capture" or "acquire" the input signal.
  • The range of input frequencies over which the PLL can successfully acquire a lock is called the Capture Range (\(2f_c\)). It is always smaller than or equal to the lock range.
  • Capture range depends on the bandwidth of the low pass filter.
3. Lock Mode (2 Marks)
  • The output signal frequency of VCO changes continuously until it becomes exactly equal to the input signal frequency (\(f_{out} = f_{in}\)).
  • At this point, the PLL is said to be in the lock mode (also called the tracking mode).
  • Once locked, the phase difference between \(f_{in}\) and \(f_{out}\) is constant and the PLL tracks any further variations in the input signal.
  • The range of input frequencies over which the PLL can maintain lock (once already acquired) is called the Lock Range (\(2f_L\)).
  • Lock Range is always greater than or equal to the Capture Range: \( 2f_L \geq 2f_c \).
  • If the input frequency goes outside the lock range, the PLL will lose lock and go back to the free running mode.
Q.3 List different types of phase detectors. Explain any one. 6 Marks
Types of Phase Detectors (1 Mark)

There are 3 types of phase detectors used in PLL:

  1. Ex-OR phase detector (Digital type)
  2. Edge triggered phase detector (Digital type)
  3. Monolithic phase detector (Analog type)
a) EXOR Phase Detector (2.5 Marks)
Fig(a): EX-OR Phase Detector

Figure shows the digital type XOR phase detector. It uses CMOS type 4070 Quad 2-input XOR gate. The output of the XOR gate is HIGH when only one of the input signals \(f_s\) (or) \(f_o\) is HIGH. This type of detection is used when both the input signals are square waves.

Fig(b): Input & Output Waveforms

The input and output waveforms for \(f_s = f_o\) are shown in fig(b). In this figure \(f_s\) is leading \(f_o\) by \(\phi\) degrees.

Fig(c): DC Output Voltage Vs Phase Difference Φ

It can be seen that the maximum dc output voltage occurs when the phase difference is \(\pi\) because the output of the gate remains high throughout.

The slope of the curve gives the conversion ratio \(K_\phi\) of the phase detector. So the conversion ratio \(K_\phi\) for a supply voltage \(V_{cc} = 5V\) is:

\( K_\phi = \dfrac{V_{cc}}{\pi} = \dfrac{5}{\pi} = 1.59 \) V/rad

b) Edge Triggered Phase Detector (2.5 Marks)
Fig(a): Edge Triggered Phase Detector (R-S Flip Flop using NOR gates)

The edge triggered digital phase detector is shown in figure (a). The circuit is an R-S flip flop made by NOR gates. This circuit is useful when \(f_s\) and \(f_o\) are both pulse waveforms with duty cycle less than 50%.

Fig(b): Input and Output Waveforms

The output of the R-S flip flop changes its state on the leading edge of \(f_s\) and \(f_o\) as shown in fig(b). The variation of dc output voltage vs \(\phi\) is shown in fig(c).

Fig(c): DC Output Voltage Vs Φ (Linear up to 2π)

This type of detector has better capture, tracking and locking characteristics as the dc output voltage is linear up to 360° compared to only 180° in the case of the EX-OR detector. Therefore, the conversion ratio is:

\( K_\phi = \dfrac{V_{cc}}{2\pi} \)

Pin Diagram of IC 565 (14-Pin DIP)
Pin Descriptions (3 Marks)

IC 565 is a 14 pin IC, operated from a dual power supply +V (at pin no. 10) and -V (at pin no. 1).

  • Pin no 1: Negative supply voltage (-Vcc). This is the ground reference for the dual supply configuration.
  • Pin no 2 & 3: Signal input for the phase detector. The input signal whose frequency is to be tracked is applied between these two pins.
  • Pin no 4: VCO output is available at this pin. This is the square wave output of the internal VCO.
  • Pin no 4 & 5: These are shorted externally so that the VCO output is applied for phase detection. In some applications, the PLL loop is broken and some external circuit is connected between pin no 4 and 5.
  • Pin no 6: Reference dc voltage is available. This is used as a bias reference internally.
  • Pin no 7: Demodulated output / VCO control voltage. If the input signal between pin no 2 and 3 is an FM signal, then at pin no 7 we get the FM demodulated output.
  • Pin no 8 and 9: External timing components R1 and C1 for VCO are connected here. They determine the free running frequency of the VCO using: \( f_0 = \dfrac{0.3}{R_1 C_1} \)
  • Pin no 10: Positive supply voltage (+Vcc).
  • Pin no 11-14: NC (No Connection). These pins are not used internally.
Q.5 Explain features of IC 565 6 Marks
Features of IC 565 PLL (Write 1 point = 1 mark approx.)

The key features and specifications of the Phase Locked Loop IC 565 are:

  • 1. Power Supply Range: It operates over a wide power supply voltage range of ±5V to ±12V (dual supply). This makes it versatile for use in different circuit configurations.
  • 2. Wide Operating Frequency: The VCO free running frequency can be adjusted over a very wide range, from 0.001 Hz to 500 kHz, by selecting appropriate values of external timing components R1 and C1. The free running frequency is given by: \( f_0 = \dfrac{0.3}{R_1 C_1} \).
  • 3. High Frequency Stability: The center frequency of the VCO is highly stable, with a typical temperature drift of only 200 ppm/°C (parts per million per degree Celsius). This ensures reliable operation over temperature variations.
  • 4. Adjustable Bandwidth: The bandwidth of the PLL (i.e., capture range and lock range) can be easily adjusted by changing the value of an external filter capacitor connected between pin 7 and the supply.
  • 5. Triangle Wave Output: A highly linear triangle wave output is available at pin 9 (timing capacitor). This can be used for waveform generation applications.
  • 6. FM Demodulation: The IC provides excellent linearity for FM demodulation applications. When an FM signal is applied at the input, the demodulated audio signal is available at pin 7.
  • 7. TTL/DTL Compatibility: The square wave output of the VCO (pin 4) is TTL and DTL compatible, making it easy to interface with digital circuits.
  • 8. Applications: Frequency multiplier, frequency synthesizer, FM demodulation, motor speed control, tone decoding, FSK (Frequency Shift Keying) demodulation, and signal conditioning.
